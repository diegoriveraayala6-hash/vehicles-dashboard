# Vehicles Dashboard

## Descripción
Aplicación web interactiva para el análisis exploratorio de anuncios de venta de coches en Estados Unidos.

## Funcionalidad
- Histograma de la distribución del odómetro
- Gráfico de dispersión de precio vs odómetro

## Tecnologías
- Python
- Streamlit
- Pandas
- Plotly Express

## Cómo ejecutar
1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar la app: `streamlit run app.py`
